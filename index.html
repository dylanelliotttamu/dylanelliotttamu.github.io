My Projects

Here are some recent projects I've worked on:

  • Integrated Approach to Augment a NWP Model

    In this research project, a machine learning (ML) training method called the iterative training technique was tested on a global circulation model. Traditionally, the process of data assimilation and the training of a ML model are kept separate. However, this technique integrates these two processes together, by training the next iteration of a hybrid model (one that has a ML component and a knowledge-based component) on the last analyses it produced. We found promising results using this technique and the work is still ongoing. At Texas A&M, I developed skills in high performance computing, data analysis, problem solving strategies, and a deep understanding of numerical weather prediction models, python, and FORTRAN. I am on track to defend my MS thesis in Fall 2024 and have a manuscript submitted to a top journal of the atmospheric sciences based on the results.

  • Independent Data Analyst for CNC

    At Catastrophe & National Claims, I was responsible for developing a program to efficiently pull elevation data from lidar point cloud datasets via prescribed coordinates from scratch. The program was designed to systemically process the significant amount of data required while not exceeding the performance capabilities of a standard laptop.

    This program was also engineered to be easily adapted to another project. I was able to successfully build this program using only open source software. As a leader in data collection, I mentored and provided oversight for new data analysts on data collection techniques by providing written feedback reports and informal input on a daily basis.

  • Developed Wavepoolweather.com

    I developed a fully-automated dynamic website to forecast wave pool water temperatures at WavePoolWeather.com. In Waco, TX lies arguably the nation's most renowned wave pool for surfing, Waco Surf. Here there is a demand to accurately predict the water temperature because knowing the water temperature is important to deciding what type of wetsuit you need (if one at all). This is especially important for the fall and spring as the weather in Texas is extremely erratic during these seasons.

    To do this, I spun up a virtual Linux machine using an AWS EC2 instance and Apache to host the server. I coded the backend to read national weather service APIs and manage files automatically using python scripts scheduled by cronjobs. I designed the front end using CSS and HTML. I plan to create forecasts for every public wave pool in the world in the coming weeks.
